# Class Intent Contracts README

This directory contains **Intent Contracts** for non-trivial classes.

An Intent Contract is a human-readable declaration of what a class is meant to do, what it must never do, and how it fails.

---

## When Is an Intent Contract Required?

This repo uses functions rather than complex classes. Intent contracts are not currently required. This infrastructure exists in case the repo grows to include stateful classes.

---

## Active Contracts

None. The repo's modules are function-based.

---

## Governance

Intent contracts, if created, are governed by ADR-003 (Authority of Declarations) and ADR-005 (Testing).
